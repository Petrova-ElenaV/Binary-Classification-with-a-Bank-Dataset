# Binary-Classification-with-a-Bank-Dataset
# Описание проекта	
Данная задача решалась мной в рамках соревнований от Kaggle Playground Series - Season 5, Episode 8:
https://www.kaggle.com/competitions/playground-series-s5e8
В исходном датасете по каждому клиенту банка приведены  данные по клиентам банка, а также данные по проведенной маркетинговой компании. 
# Цель
Требуется построить модель машинного обучения, которая на основе предложенных характеристик клиента и маркетинговой компании будет предсказывать, воспользуется он предложением об открытии депозита или нет.
# Ключевые результаты	
1. Получен roc_auc = 0.97501
2. Заняла 515 место из 3367 участников
3. Построила модель на основе LGBM с подбором гиперпараметров
4. Провела feature engineering и анализ важности признаков
# Стек	
OneHotEncoder, LabelEncoder, CountEncoder, LGBMClassifier
